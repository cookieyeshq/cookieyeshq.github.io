Quick Setup

1

Install the SDK

Add ConsentFlow to your project using CocoaPods or Gradle

2

Initialize

Configure the SDK with your Firebase project settings

3

Collect Consent

Show consent dialogs and manage user preferences automatically

// iOS (Swift)
import ConsentFlow
import FirebaseAnalytics

ConsentFlow.configure(
  firebaseApp: FirebaseApp.app()
)

ConsentFlow.showConsentDialog()
// Android (Kotlin)
import com.consentflow.sdk.ConsentFlow

ConsentFlow.initialize(this)
ConsentFlow.requestConsent()
